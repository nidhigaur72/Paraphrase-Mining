# 🧠 Paraphrase Mining with Sentence Transformers

This project demonstrates **Paraphrase Mining** and **Semantic Text Similarity** using the **Sentence Transformers** library. It shows how to identify semantically similar sentences, compute cosine similarity, detect paraphrases, and identify outlier sentences using pre-trained transformer models.

---

## 📌 Project Overview

The notebook explores Natural Language Processing (NLP) techniques for comparing sentence meanings instead of just matching words. It includes:

- Generating sentence embeddings
- Finding paraphrase pairs
- Measuring semantic similarity
- Detecting sentence outliers
- Using multilingual transformer models

---

## 🚀 Features

- ✅ Paraphrase mining using `SentenceTransformer`
- ✅ Semantic similarity with cosine similarity
- ✅ Sentence embedding generation
- ✅ Outlier sentence detection
- ✅ Multilingual sentence comparison
- ✅ Easy-to-understand implementation in Jupyter Notebook
